🛍️ Customer Segmentation using K-Means Clustering

📌 Overview

This project implements the K-Means Clustering algorithm to group customers into distinct segments based on their purchasing behavior. Customer segmentation helps businesses better understand their customers and develop targeted marketing strategies.

🎯 Objective

The objective of this project is to:
- Understand unsupervised machine learning.
- Apply the K-Means clustering algorithm.
- Identify meaningful customer groups.
- Visualize clusters for better interpretation.
- Gain insights into customer purchasing patterns.

🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

📂 Dataset

The project uses the Mall Customer Segmentation Dataset from Kaggle.

Features commonly used:
- Annual Income (k$)
- Spending Score (1–100)

⚙️ Project Workflow

1. Import the dataset.
2. Select relevant features.
3. Perform data preprocessing.
4. Determine the optimal number of clusters using the Elbow Method.
5. Train the K-Means clustering model.
6. Assign customers to different clusters.
7. Visualize the customer segments.

📊 Results

The model groups customers into different clusters based on their spending habits and annual income, enabling better customer analysis and targeted business strategies.

📈 Output

The project generates:
- Elbow Method graph
- Customer segmentation scatter plot
- Cluster centroids
